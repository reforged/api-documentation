# Utilisateurs

Les Utilisateurs sont des personnes qui pocede un compte et qui peuvent donc utiliser notre application. Sur cette page nous verront comment créer et modifier un compte.

## Le modèle Utilisateurs

Le modèle Utilisateurs contient toutes les informations relatives aux utilisateurs.C'est a dire l'email et le mot de passe pour n'en citer que quelque un. En outre, les Utilisateurs peuvent également être reliés aux questions et sequences qu'ils ont créé ainsi qu'a un roles et une permission.

### Properties

  </Property>
  <Property name="updated_at" type="timestamp">

  </Property>
</Properties>
